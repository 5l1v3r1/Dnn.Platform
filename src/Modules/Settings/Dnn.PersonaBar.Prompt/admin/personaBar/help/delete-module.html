delete-module

Soft-deletes a module on a specific page. The module will be sent to the DNN Recycle Bin. This will not uninstall modules or affect module definitions.

Options

Flag Value Required Default Value Description
--id integer Yes Explicitly specifies the Module ID of the module to retrieve. Use of the flag is not required. You can simply provide the ID value as the first argument.
--pageid integer Yes Specifies the Page ID of the page on which the module to delete resides.

Delete and Send Module Instance to Recycle Bin

This will delete a module instance on a specific page and send it to the DNN Recycle Bin

delete-module 345 --pageid 42
