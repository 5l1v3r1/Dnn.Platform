purge-user

Permanently deletes the specified user from the portal. The user must be deleted already. If you issue a get-user command and the IsDeleted property isn't true, then you will get an error when attempting this command. You must use the delete-user command on the user first.

Options

Flag Type Required Default Description
--id integer Yes The user's User ID. If the flag is not used, then the user's ID must be the first argument.

Permanently Delete a User

Permanently delete's the user with a User ID of 345. If you issue the command: get-user 345 you will receive a 'user not found' message.

purge-user 345

This is the more explicit form of the above code.

purge-user --id 345
