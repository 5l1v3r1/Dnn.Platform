purge-page

Permanently deletes a page from the portal that had previously been deleted and sent to DNN's Recycle Bin.

Options

Flag Value Required Default Description
--id integer Yes Explicitly specifies the Page ID to purge. Use of the flag name is not required. You can simply provide the ID value as the first argument.
--deletechildren bool No false Specifies that if a page has children, should the command delete them all or show error.

Purge a Deleted Page By Page ID

purge-page 999 OR purge-page --id 999

Purge a Deleted Page and All It's Child Pages

purge-page --id 999 --deletechildren true
