set-task

Set or update properties on the specified Scheduled task.

Options

Flag Type Required Default Description
--id integer Yes The Schedule ID of the task you want to update. You can avoid explicitly typing the --id flag by just passing the Schedule ID as the first argument.
--enabled boolean No When true, the specified task will be enabled. When false, the specified task will be disabled.

Disable the "Purge Schedule History" Task

set-task 3 --enabled false OR set-task --id 3 --enabled false
