copy-module

Copies a module to the specified page

Options

Flag Value Required Default Value Description
--id integer Yes Explicitly specifies the Module ID of the module to copy. Use of the flag is not required. You can simply provide the ID value as the first argument.
--pageid integer Yes The Page ID of the page that contains the module you want to copy.
--topageid integer Yes The Page ID of the target page. The page to which you want to copy the module.
--includesettings boolean No true If true, Prompt will copy the source module's settings to the copied module.

Copy a Module from One Page to Another

This command makes a copy of the module with Module ID 358 on the Page with Page ID of 71 and places that copy on the page with a Page ID of 75

copy-module 358 --pageid 71 --topageid 75

Results

ModuleId : 358
Title : My Module
ModuleName : DNN_HTML
FriendlyName : HTML
ModuleDefId : 120
TabModuleId : 107
AddedToPages : 71, 75
Successfully copied the module.
