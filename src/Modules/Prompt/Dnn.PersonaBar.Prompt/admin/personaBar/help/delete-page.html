delete-page

Deletes a page within the portal and sends it to the Recycle Bin. For added safety, this method will not allow deletion of pages with children, Host pages, or pages in the //Admin path.

Options

Flag Value Required Default Description
--parentid string See Notes When you want to delete all child pages of a certain page, use this flag and specify the parent's Page ID. Required if --id is not specified.
--id integer See Notes Explicitly specifies the Page ID to delete. Use of the flag name is not required. You can simply provide the ID value as the first argument. Required if --parentid is not specified.

Delete Page By Page ID

delete-page 999

Delete All First-Level Children of Parent Page

NOTE: At this time, Prompt will not allow you to delete any pages that have children

delete-page --parentid 999
