add-roles

Add one or more DNN security roles to a user

Options

Flag Type Required Default Description
--id integer true User ID of user to which the roles will be added. If a number is passed as the first argument, you do not need to use the --id flag explicitly
--roles string true Comma-delimited string of DNN role names to apply to user.

Add a Role to a User

add-roles --id 23 --roles Editor OR add-roles 23 --roles Editor OR add-roles --id 23 "Editor"

Add a Multi-Word Role to a User

add-roles --id 23 --roles "Article Reviewer"

Add Multiple Roles to a User

add-roles --id 23 --roles "Editor, Writer, Article Reviewer"
