new-page

Creates a new page within the portal

Options

Flag Value Required Default Description
--name string Yes The name to use for the page. The --name flag does not have to be explicitly declared if the first argument is a string and not a flag.
--title string No The display title to use for the page
--url string No A custom URL to use for the page. If not specified, the --name is used by DNN when creating the URL
--parentid integer No If you want this to be the child of a page, specify that page's ID as the --parentid
--description string No The description to use for the page
--keywords string No Keywords to use for the page
--visible boolean No true If true, the page will be visible in the site's navigation menu

Create a New Page (Minimum Syntax)

Implicitly Use --name flag
new-page "My New Page"
Explicitly Use --name flag
new-page --name "My New Page"
TabId:78
Name:My New Page
Title:

Description:

Create a Child Page with Additional Options

new-page --name "My Sub Page" --parentid 78 --title "This is the sub page title" --keywords "sample, sub-page, prompt, KnowBetter"
