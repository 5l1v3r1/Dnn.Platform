purge-page

Permanently deletes a page from the portal that had previously been deleted and sent to DNN's Recycle Bin.

Options

Flag Value Required Default Description
--id integer See Notes Explicitly specifies the Page ID to purge. Use of the flag name is not required. You can simply provide the ID value as the first argument. Required if --parentid and --name are not specified.
--name string See Notes Specifies the name (not title) of the page that should be purged. This can be combined with --parentid to target a page name with a specific Parent page. Required if --parentid and --name are not specified.
--parentid string See Notes When you want to purge all child pages of a certain page, use this flag and specify the parent's Page ID. Required if --id and --name are not specified.

Purge a Deleted Page By Page ID

purge-page 999 OR purge-page --id 999

Purge a Deleted Page With A Specific Page Name

purge-page --name "Page1"

Purge All Deleted Pages With A Common Parent Page

purge-page --parentid 999
