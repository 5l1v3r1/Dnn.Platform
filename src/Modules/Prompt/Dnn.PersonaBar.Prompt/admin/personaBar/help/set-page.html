set-page

Sets or updates properties of the specified page

Options

Flag Value Required Default Description
--id integer No Current Page ID The ID of the page you want to update. If not specified, the command will operate on the current page.
--name string No The name to use for the page
--title string No The display title to use for the page
--url string No A custom URL to use for the page. If not specified, the --name is used by DNN when creating the URL
--parentid integer No If you want this to be the child of a page, specify that page's ID as the --parentid
--description string No The description to use for the page
--keywords string No Keywords to use for the page
--visible boolean No If true, the page will be visible in the site's navigation menu

Update Properties on Current Page

set-page --title "My New Page Title" --name Page25
