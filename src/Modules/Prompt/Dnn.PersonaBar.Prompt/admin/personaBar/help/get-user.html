get-user

Enables you to display the current user's information, or retrieve a user by their User ID, Username or Email address. Supports partial searching. Returns the first user found.

Options

Flag Value Required Default Description
--id integer No Explicitly specifies the UserId for the user being retrieved.
--email string No The email address for the user being retrieved. You can use the asterisk ( * ) as a wildcard to signify 0 or more characters.
--username string No The username for the user being retrieved. You can use the asterisk ( * ) as a wildcard to signify 0 or more characters.

Get Current User

get-user

Get User by User ID

Implicit Use of --id flag

When specifying a single value after the command name, if it is an integer, Prompt will assume it is a User ID

get-user 345
Explicit use of --id flag

You can explicitly use the --id flag to avoid any confusion

get-user --id 345
UserId:345
Username:jsmith
Email:jsmith@sample.com
DisplayName:John Smith
FirstName:John
LastName:Smith
LastLogin:2016-12-06T09:31:38.413-07:00
IsAuthorized:true

Get User by Email

Implicit Use of --email flag

When specifying a single value after the command name, if it contains the at symbol ( @ ), Prompt will assume it is an Email

get-user jsmith@sample.com
Explicit Use of --email flag

You can explicitly use the --email flag to avoid any confusion

get-user --email jsmith@sample.com
UserId:345
Username:jsmith
Email:jsmith@sample.com
DisplayName:John Smith
FirstName:John
LastName:Smith
LastLogin:2016-12-06T09:31:38.413-07:00
IsAuthorized:true

Search for User by Email Using Wildcards

Implicit Use of --email flag

When specifying a single value after the command name, if it contains the at symbol ( @ ), Prompt will assume it is an Email. Only the first matched user record will be returned. Try using list-users if you would like to find all matching users.

get-user jsmith*@sample.com
Explicit Use of --email flag
get-user --email jsmith*@sample.com
UserId:345
Username:jsmith
Email:jsmith@sample.com
DisplayName:John Smith
FirstName:John
LastName:Smith
LastLogin:2016-12-06T09:31:38.413-07:00
IsAuthorized:true

Get User by Username

Implicit Use of --username flag

When specifying a single value after the command name, if it is not a number and does not contain the at symbol ( @ ), Prompt will assume it is a Username

get-user jsmith
Explicit Use of --username flag

You can explicitly use the --username flag to avoid any confusion

get-user --username jsmith
UserId:345
Username:jsmith
Email:jsmith@sample.com
DisplayName:John Smith
FirstName:John
LastName:Smith
LastLogin:2016-12-06T09:31:38.413-07:00
IsAuthorized:true

Search for User by Username Using Wildcards

Implicit Use of --username flag

get-user --username jsmith*
