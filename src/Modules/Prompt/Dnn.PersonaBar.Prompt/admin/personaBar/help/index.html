Prompt Commands

Here is a list of available commands for Prompt.

Commands

Command Description
General Commands
help Retrieves this list of available commands
clh Shortcut for clear-history.
cls Shortcut for clear-screen.
clear-history Empties the list of Prompt commands you have previously executed in this session.
clear-screen Clears the Prompt console window, giving you a blank screen.
config Configures the Prompt console.
goto Navigates the browser to the specified page.
reload Causes the browser to reload the page.
set-mode Sets the DNN View Mode. For example: Edit or Layout.
Page Commands
get-page Retrieves details about the specified page.
list-pages Retrieves a list of pages based on the specified criteria.
new-page Creates a new page within the portal.
set-page Sets or updates properties of the specified page.
delete-page Sends a page to the DNN Recycle Bin.
restore-page Recovers a page from the DNN Recycle Bin.
purge-page Permanently deletes a page from the DNN Recycle Bin.
User Commands
get-user Retrieves details about the specified user account.
list-users Retrieves a list of users based on the specified criteria.
new-user Creates a new user record in the portal
set-user Enables updating user properties and values such as Username, Email, etc.
delete-user Deletes the specified user, but the user can still be recovered.
purge-user Permanently deletes a user that has been marked as 'deleted'.
restore-user Recovers a user that is marked as 'deleted' but has not been purged.
add-roles Add one or more DNN security roles to a user.
reset-password Sends a reset password email to the user.
Module Commands
copy-module Copies a module to the specified page.
delete-module Permanently deletes a module instance and all referenced instances, or soft-deletes a module on a specific page.
list-modules Retrieves a list of modules based on the specified criteria.
move-module Moves a module to the specified page.
get-module Retrieves details about a single module.
restore-module recovers a module from the DNN Recycle Bin.
Host Commands
clear-cache Clears the server cache and reloads the page.
get-host Retrieves information about the DNN installation.
new-extension Installs an extension in the DNN instance.
restart-application Restarts the DNN instance.
Portal Commands
clear-log Clears the Event Log for the current portal.
get-portal Retrieves information about the current portal.
Role Commands
get-role Retreives details about the specified DNN security role.
delete-role Permanently deletes the given DNN security role.
list-roles Retrieves a list of DNN security roles for the portal.
new-role Creates a new DNN security role for the portal.
set-role Sets or updates properties of the specified DNN security role.
Scheduler Commands
get-task Retrieves details for the specified DNN Schedule item.
list-tasks Retrieves a list of scheduler items based on the specified criteria.
set-task Sets or updates properties of the specified scheduled task.
