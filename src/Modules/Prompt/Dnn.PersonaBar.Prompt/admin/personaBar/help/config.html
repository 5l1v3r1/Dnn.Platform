config

Configures the Prompt console.

Options

Flag Type Required Default Description
--height CSS Unit Expression No Specifies the height of the Prompt console in CSS units (200px, 50%, 30pt, etc.). If you only pass one argument to the config command, Prompt will assume that argument is the height. In that case you do not need to explicitly use --height

Set Console Height

config 50%
[Height of Console gets set to 50% of the window height.
