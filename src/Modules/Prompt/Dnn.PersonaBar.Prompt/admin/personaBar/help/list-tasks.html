list-tasks

Retrieves a list of scheduled tasks based on the specified criteria. DNN refers to these as schedules or scheduler items. Prompt refers to them as tasks.

Options

Flag Type Required Default Description
--enabled boolean No When specified, Prompt will return tasks that are enabled (if this flag is set to true) or disabled (if this is set to false). If this flag is not specified, Prompt will return tasks regardless of their enabled status.
--name string No When specified, Prompt will return tasks whose Friendly Name matches the expression. This supports wildcard matching via the asterisk ( * ) to represent zero (0) or more characters.
