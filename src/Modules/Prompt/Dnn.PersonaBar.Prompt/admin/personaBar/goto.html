Navigates to the specified page within the DNN portal

Options

Flag Type Required Default Descriptions
--id integer Yes Specify the Page (Tab) ID of the page to which you want to navigate. Explicit use of the --id flag is not needed. Simply pass in the Page ID as the first argument after the command name

Navigate to A Page Within the Site
